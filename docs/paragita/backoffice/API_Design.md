# API Design
## Paragita
Endpoint should form:
`$WEBSITE/api/$API_VER/$ENDPOINT`

Currently:

| Variable | Value |
| ------------- | ------------- |
| $WEBSITE | http://devel.paragita.com |
| $API_VER | v1.0 |

Modules available:

| Module | $ENDPOINT | Desc|
| ------------- | ------------- |-|
| Member Management | member | Member management|
| Event Management | event | Rehearsal, concert, job|
